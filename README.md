# EVOX — 三层进化引擎

> Darwin + EvoPrompt + SEW | 让所有项目自主进化

## 架构
- **Darwin层**：代码质量结构化审计（工程层进化）
- **EvoPrompt层**：Prompt自动优化（AI层进化）
- **SEW层**：Self-Evolving Workflow（工作流进化）

## Quick Start
```bash
cd evox
pip install -r requirements.txt
python -m evox
```

## 📐 理论基础
> **Harness理论**：在AI领域，Harness（环境设计）比模型本身更重要。
> **红杉论点**：从卖工具到卖结果。EVOX交付持续进化的AI系统，而非进化工具。
