# EvoX

三层进化引擎 — Darwin + EvoPrompt + SEW | 让所有项目自进化

EvoX 不卖进化算法，卖项目自进化能力。接入任意AI项目，自动评估改进验证，只保留有效改进。

## 三层架构

- Layer 1 Darwin: 适者生存，多项目矩阵自然选择
- Layer 2 EvoPrompt: 棘轮机制，评估即进化不可回滚
- Layer 3 SEW: 归海层，自动捕获外部改进融入自身

## 进化成果

6项目矩阵平均提升19分，零回滚。

## 快速开始

    git clone https://github.com/MoKangMedical/evox.git
    cd evox
    pip install -r requirements.txt
    python src/darwin.py --project medichat-rd --rounds 5
